# `@IBAction`
인터페이스 빌더에 있는 UI Component가 어떤 이벤트를 발생했을 때 실행되는 액션을 함수에 연결해주는 키워드

```swift
@IBAction func touchUpResetButton(_ sender: UIButton) {
    ...
}
```
